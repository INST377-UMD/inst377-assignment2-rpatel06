Voice Commands

You can use the following voice commands on this website:

  • "Hello" - Will display a Hello World alert
  • "Change the color to [color]" - Changes the background color
  • "Navigate to [page]" - Navigate to Home, Stocks, or Dogs page
  • "Lookup [stock]" - Looks up stock data for the specified ticker
